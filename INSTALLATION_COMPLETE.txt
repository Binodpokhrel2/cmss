🚀 NEXT STEPS (FOLLOW THESE):

1. UPLOAD FILES TO YOUR SERVER
   - Upload all files to your web hosting (public_html, www, or htdocs)
   - Keep the same folder structure
   - Set file permissions: chmod 644 for files, chmod 755 for directories

2. CREATE THE DATABASE
   - Login to phpMyAdmin or MySQL client
   - Create new database: "canteen_management"
   - Import database.sql file
   OR use command line:
   $ mysql -u root -p < database.sql

3. CONFIGURE DATABASE CONNECTION
   - Edit config.php
   - Update these lines with your credentials:
     define('DB_HOST', 'localhost');    // Your MySQL host
     define('DB_USER', 'root');         // Your MySQL username
     define('DB_PASS', '');             // Your MySQL password
     define('DB_NAME', 'canteen_management');

4. SET FILE PERMISSIONS (Linux/Mac)
   $ chmod -R 755 /path/to/canteen
   $ chmod 644 /path/to/canteen/*.php
   $ chmod 644 /path/to/canteen/css/style.css
   $ chmod 644 /path/to/canteen/js/script.js

5. VERIFY INSTALLATION
   - Open http://your-domain.com/ in a browser
   - Homepage should load
   - Click "Get Started" → Login with credentials below
   - Explore admin and staff dashboards

📞 TROUBLESHOOTING:

Problem: "Connection failed" error
Solution: Check DB_HOST, DB_USER, DB_PASS in config.php
          Ensure MySQL is running
          Verify database exists

Problem: Login fails
Solution: Check database has users table with demo accounts
          Clear browser cookies
          Verify config.php settings

Problem: CSS/JS not loading
Solution: Check file paths match your directory structure
          Clear browser cache
          Verify file permissions

Problem: Pages not found
Solution: Check all PHP files are uploaded
          Verify folder structure matches
          Check web server PHP handling

See README.md for more troubleshooting tips.
